zettwerk.mobile
===============

Apply jquery.mobile based theme to plone.


Usage
=====

Add zettwerk.mobile to your buildout and install it via quickinstaller. It also installs zettwerk.mobiletheming for url based theme switching. Go to the plone control panel to mobile theming and set up a hostname, under which the theme should be applied. Also choose zettwerk.mobile from the list of existing themes as mobile theme.

See it in action: https://www.youtube.com/watch?v=Q2ID86XkiQQ
